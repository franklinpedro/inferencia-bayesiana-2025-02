# Inferência Bayesiana

Este repositório reúne os materiais de laboratório da disciplina **Inferência Bayesiana** (2025-02) do curso de Estatística da Universidade Federal de Uberlândia (UFU).

O foco aqui é a parte **prática** da disciplina: scripts em R, notebooks/arquivos Quarto e conjuntos de dados usados nas atividades de laboratório.

> **Obs.:** Neste primeiro momento, o repositório contém apenas materiais dos laboratórios. Outros recursos (slides, listas, etc.) poderão ser adicionados futuramente.
